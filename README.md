# php_mysql_ai2
## 此APP為模型店資料庫範例
### 1.將test資料夾裡面的程式碼下載後放在此目錄底下才能運作，目錄為 C:\AppServ\www
### 2.執行之前需要把myschool.sql先匯入資料庫裡才能找到students 的資料
### 3.程式碼執行請用瀏覽器輸入網址： 127.0.0.1/xxx.php  或是 127.0.0.1/xxx.php?sql=select * from students
### 4.並且把對應的欄位名稱填入sql語法才能正確顯示，mysql.pdf有sql語法教學說明
### 5.APP上下頁的每一頁都只有顯示3筆資料
